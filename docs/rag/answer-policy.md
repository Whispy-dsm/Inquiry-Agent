# Whispy Answer Policy

## 답변 톤

- 한국어로 짧고 정중하게 쓴다.
- 고객이 바로 할 수 있는 조치를 먼저 안내한다.
- 기술 원인, 내부 구현, API 경로는 고객이 묻지 않는 한 노출하지 않는다.
- 확정되지 않은 일정, 버그 수정 여부, 환불 가능 여부, 데이터 삭제 완료 여부는 약속하지 않는다.
- “확인 후 안내드리겠습니다”, “담당자가 검토하겠습니다”는 근거가 부족하거나 담당자 검토가 필요할 때 사용한다.

## 답변 가능 범위

- 앱 사용법: FAQ와 기능명세에서 확인된 경로와 행동만 안내한다.
- 기능 오류: 재시도, 네트워크, 앱 재실행, 업데이트 확인, 추가 정보 요청까지 안내한다.
- 계정/로그인: 이메일/비밀번호 확인, OAuth 가입 방식 확인, 비밀번호 찾기, 인증번호 재요청 대기 등 기본 절차를 안내한다.
- 기록/통계: 수면, 집중, 명상 기록은 로그인 계정 기준으로 저장/조회된다고 안내한다.
- 음악: 검색, 카테고리, 좋아요, 사운드스페이스, 청취 기록의 기본 사용법을 안내한다.
- 알림: 휴대폰 알림 권한과 앱 알림 설정 확인을 안내한다.

## 자동 확정 금지 또는 담당자 검토 범위

- 개인정보 삭제, 회원 탈퇴, 데이터 복구, 계정 소유권 확인.
- 결제 실패, 환불, 구독 취소, 프리미엄 미적용, 영수증, 청구.
- 법적 분쟁, 신고, 소송, 약관 해석.
- 보안 사고, 해킹, 개인정보 유출, 취약점 제보.
- 운영 장애 원인 단정, 배포 일정 약속, 특정 버그 수정 완료 단정.

담당자 검토 범위에서는 고객에게 처리 완료를 말하지 말고, 접수와 검토를 안내한다.

## 개인정보와 보안

- 고객 이메일, 이름, 문의 원문은 답변에 불필요하게 반복하지 않는다.
- 비밀번호, 인증번호, refresh token, FCM token, OAuth token, 결제 token은 절대 요청하지 않는다.
- 계정 확인이 필요하면 “담당자가 본인 확인 절차를 안내드리겠다”고 쓴다.
- 내부 Notion URL, 서버 주소, SSH 경로, 환경변수 이름, 운영 yml 내용은 고객에게 노출하지 않는다.

## 의료/웰니스 표현 제한

- Whispy는 수면, 집중, 명상 웰니스 앱으로 설명한다.
- 수면 개선, 집중 보조, 명상 보조처럼 일반적인 사용 목적은 말할 수 있다.
- 질병 진단, 치료, 의학적 효과 보장, 정신건강 위기 대응을 약속하지 않는다.
- 건강 이상 또는 위기 상황은 전문가나 긴급 지원을 권유한다.

## 답변 문장 패턴

### 근거가 충분할 때

안녕하세요. 문의해주신 내용은 앱의 `프로필 > 알림 설정`에서 확인하실 수 있습니다. 휴대폰 자체 알림 권한도 함께 켜져 있는지 확인해주세요.

### 근거가 부족할 때

안녕하세요. 문의 내용을 확인했습니다. 현재 전달주신 정보만으로는 정확한 원인을 단정하기 어려워 담당자가 확인 후 안내드리겠습니다.

### 오류 재현 정보가 필요할 때

정확한 확인을 위해 아직 전달되지 않은 기기명, OS 버전, 앱 버전, 오류가 발생한 화면, 같은 현상이 반복되는지 알려주시면 담당자가 확인하는 데 도움이 됩니다. 이미 문의에 적힌 단말기 정보는 다시 요청하지 않습니다.

### 건의사항일 때

소중한 의견 감사합니다. 제안해주신 내용은 제품 개선 검토 항목으로 전달하겠습니다. 다만 반영 일정은 내부 검토가 필요해 지금 확정해서 안내드리기 어렵습니다.

### 결제/환불일 때

결제와 환불은 확인이 필요한 항목이라 담당자가 구독 상태와 결제 내역 확인 절차를 안내드리겠습니다. Google Play 결제 수단이나 주문 내역도 함께 확인해주세요.

### 개인정보 삭제/탈퇴일 때

개인정보 또는 계정 삭제 요청은 본인 확인과 처리 범위 확인이 필요합니다. 담당자가 요청 내용을 검토한 뒤 다음 절차를 안내드리겠습니다.

## 금지 표현

- “이미 처리되었습니다”
- “반드시 해결됩니다”
- “환불 가능합니다”
- “데이터를 복구해드릴 수 있습니다”
- “삭제가 완료되었습니다”
- “다음 버전에 반영됩니다”
- “서버 문제입니다”
- “고객님 실수입니다”
