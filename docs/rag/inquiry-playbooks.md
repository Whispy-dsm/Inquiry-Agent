# Whispy Inquiry Playbooks

## 문의 유형

| Google Form 내부 유형 | 의미 | 기본 위험도 | 기본 처리 |
| --- | --- | --- | --- |
| `APP_ERROR` | 앱 오류/버그/멈춤/로그인 실패 | low 또는 키워드 기반 high | 재현 정보 요청 + 기본 조치 안내 |
| `SERVICE_QUESTION` | 기능 사용법/정책 문의 | low 또는 키워드 기반 high | FAQ/기능명세 기반 답변 |
| `SUGGESTION` | 개선 제안/기능 요청 | low | 접수 감사 + 내부 전달 |
| `OTHER` | 그 외 문의 | high | 사람 검토 전제 |

## 공통 초안 규칙

- 고객 이름이 있으면 첫 문장에 자연스럽게 사용할 수 있지만, 없어도 무리하게 넣지 않는다.
- “문의해주셔서 감사합니다” 뒤에 바로 핵심 안내를 쓴다.
- 해결 단계는 3개 이하로 줄인다.
- 추가 정보 요청은 필요한 경우에만 한다.
- 담당자 확인이 필요한 경우 “검토 후 안내”로 끝낸다.

## APP_ERROR

### 바로 답변 가능한 경우

- 앱이 느리거나 멈춤: 앱 완전 종료 후 재실행, 기기 재부팅, 업데이트 확인, 저장공간 확인.
- 화면이 하얗게 나옴: 네트워크 연결 확인 후 앱 재실행.
- 로그인 실패: 이메일/비밀번호 입력, 가입 방식, OAuth 로그인 버튼 확인.
- 인증번호 미수신: 스팸함, 이메일 오타, 잠시 대기 후 재요청.

### 추가로 받아야 할 정보

- 기기명
- OS 버전
- 앱 버전
- 문제가 발생한 화면
- 오류 문구 또는 스크린샷
- 재현 절차
- 네트워크 환경

### 초안 예시

안녕하세요. 문의해주신 현상은 먼저 앱을 완전히 종료한 뒤 다시 실행하고, 인터넷 연결과 앱 업데이트 여부를 확인해 주세요. 같은 문제가 반복되면 기기명, OS 버전, 앱 버전, 오류가 발생한 화면과 오류 문구를 알려주시면 담당자가 확인하겠습니다.

## SERVICE_QUESTION

### 답변 가능 주제

- 수면/집중/명상 기록 방법.
- 통계 확인 위치와 의미.
- 음악 검색, 좋아요, 사운드스페이스, 청취 기록.
- 알림 확인과 알림 설정.
- 프로필 수정, 비밀번호 찾기, 로그인 방식.
- 구독 상태 확인과 Google Play 구독 취소 경로.

### 고위험 전환 조건

- 개인정보 삭제, 탈퇴, 데이터 복구.
- 환불, 결제 오류, 구독 미적용.
- 법적 문서 해석.
- 보안 사고.

### 초안 예시

안녕하세요. 위스피에서는 음악을 검색하거나 카테고리로 찾아 들을 수 있고, 마음에 드는 음악은 좋아요로 저장할 수 있습니다. 자주 듣는 음악은 사운드스페이스에 따로 모아둘 수 있어요.

## SUGGESTION

### 답변 방향

- 제안에 감사한다.
- 제품 개선 검토 항목으로 전달한다고 안내한다.
- 반영 여부나 일정은 약속하지 않는다.
- 이미 있는 기능이면 사용 경로를 함께 안내한다.

### 초안 예시

안녕하세요. 소중한 의견 감사합니다. 제안해주신 내용은 제품 개선 검토 항목으로 전달하겠습니다. 다만 반영 여부와 일정은 내부 검토가 필요해 현재 확정해서 안내드리기 어렵습니다.

## OTHER

### 답변 방향

- 범위가 넓기 때문에 무리하게 답하지 않는다.
- 문의를 접수했음을 알린다.
- 담당자가 확인 후 안내한다고 쓴다.
- 개인정보, 결제, 법적, 보안 키워드가 있으면 반드시 high risk로 유지한다.

### 초안 예시

안녕하세요. 문의 내용을 확인했습니다. 요청하신 사항은 담당자 검토가 필요한 항목이라 확인 후 다음 절차를 안내드리겠습니다.

## 결제/구독 문의

### 안내 가능

- Google Play 결제 수단과 주문 내역 확인.
- 앱 재실행.
- 앱 내 구독 관리 또는 구매 복원 시도.
- 구독 취소는 Google Play 스토어의 결제 및 구독 메뉴에서 진행.

### 금지

- 환불 가능 여부 확정.
- 프리미엄 적용 완료 단정.
- 결제 내역을 이메일만으로 확인했다고 말하기.

### 초안 예시

안녕하세요. 결제 또는 구독 상태는 확인 절차가 필요한 항목입니다. 먼저 Google Play의 결제 수단과 주문 내역을 확인해주시고, 앱을 완전히 종료한 뒤 다시 실행해 주세요. 계속 적용되지 않으면 담당자가 구독 상태 확인 절차를 안내드리겠습니다.

## 개인정보/탈퇴 문의

### 안내 가능

- 회원 탈퇴는 앱의 프로필/설정 화면에서 진행할 수 있다.
- 탈퇴나 개인정보 삭제는 본인 확인과 처리 범위 확인이 필요하다.
- 삭제 완료는 담당자 확인 전 단정하지 않는다.

### 초안 예시

안녕하세요. 개인정보 또는 계정 삭제 요청은 본인 확인과 처리 범위 확인이 필요합니다. 요청 내용을 담당자가 검토한 뒤 다음 절차를 안내드리겠습니다.

## 데이터가 보이지 않는 문의

### 안내 가능

- 로그아웃 여부 확인.
- 다른 계정으로 로그인했는지 확인.
- 네트워크 연결 확인.
- 같은 계정으로 로그인하면 기존 기록을 다시 확인할 수 있다는 일반 안내.

### 주의

- 탈퇴 후 데이터 복구 가능성을 말하지 않는다.
- 서버에 데이터가 남아 있다고 단정하지 않는다.

## 내부 리뷰 체크리스트

- 근거 문서가 있는 답변인가?
- 고객에게 내부 구현을 노출하지 않았는가?
- 처리 완료, 환불, 삭제, 복구, 일정 확정을 약속하지 않았는가?
- 추가 정보 요청이 과하지 않은가?
- high risk 문의가 Discord 승인 없이 발송되지 않도록 표시됐는가?
